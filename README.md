# GSAP Animated Blog Website UI

## 🚀 Overview
A modern blog-style website with smooth animations built using HTML, CSS, and GSAP.

## ✨ Features
- GSAP timeline animations for navbar and hero section
- ScrollTrigger animations for each feature card
- Smooth hover effects and image transitions
- Fully responsive design
- Clean and modern UI layout

## 🛠️ Technologies Used
- HTML5
- CSS3
- JavaScript
- GSAP (GreenSock Animation Platform)

## 🎯 Animations Included
- Navbar entrance animation
- Hero text reveal animation
- Scroll-based card animations
- Left & right content transitions
- Image hover zoom & grayscale effect

## ⚡ How It Works
- GSAP Timeline handles initial page animations
- ScrollTrigger animates each `.Features` card individually
- CSS handles hover and visual styling
